☕"CoffeShop" Android App

Android application for automating sales and business processes at the "Coffee Teen" coffee & pastry shop.

✨ Roles
1. 👤 Customer
· Browse up‑to‑date menu with descriptions and prices
· Place online orders
· View order history
2. 👨‍💼 Administrator
· Manage menu items (add, edit, delete)
· Manage employee data
· Track daily sales, number of receipts, and revenue
· View popularity statistics of menu items
3. 👑 Owner
· All administrator's viewing capabilities
· Generate net profit reports
· Manage admin accounts

🛠 Tech Stack
· Language: Kotlin
· UI: Jetpack Compose + XML (Material Design)
· Backend: Firebase (Cloud Firestore, Authentication)
· IDE: Android Studio
· Testing: Firebase Test Lab + manual testing
· Security: HTTPS, Android Keystore, Firebase Security Rules

📲 Installation
1. Clone the repo:
      git clone https://github.com/AigulMiftahova/CoffeeApp
2. Open in Android Studio.
3. Add your Firebase google-services.json to the app/ folder.
4. Build and run on a device/emulator with Android 8.0+.
